Events

  • dragItemPositioned
  • fitComplete
  • layoutComplete
  • removeComplete

Packery is an Event Emitter. You can bind listeners to events with the on and off methods.

var pckry = new Packery( elem );

function onLayout() {
  console.log('layout done');
}
// bind event listener
pckry.on( 'layoutComplete', onLayout );
// un-bind event listener
pckry.off( 'layoutComplete', onLayout );
// return true to trigger an event listener just once
pckry.on( 'layoutComplete', function() {
  console.log('layout done, just this one time');
  return true;
});

dragItemPositioned

pckry.on( 'dragItemPositioned', function( pckryInstance, draggedItem ) { //...
// or with jQuery
$container.packery( 'on', 'dragItemPositioned', function( pckryInstance, draggedItem ) {} )
  • pckryInstance

    Type: Packery

    the Packery instance

  • draggedItem

    Type: Packery.Item

    the dragged item that was positioned

Triggered after a drag end, only when an item is positioned, either into a row or column, or into its container element. This event is triggered after the positioning transition has ended.

// set up Packery and Draggabillies
// like the bindDraggabillyEvents demo, then...
pckry.on( 'dragItemPositioned', function( pckryInstance, draggedItem ) {
  console.log( 'Packery positioned dragged ' + draggedItem.element.nodeName );
});

  • pckryInstance

    Type: Packery

    the Packery instance

  • item

    Type: Packery.Item

    items that was fit

Triggered after an item element is fit.

pckry.on( 'fitComplete', function( pckryInstance, item ) {
  notify( 'fit ' + item.element.className );
});

layoutComplete

pckry.on( 'layoutComplete', function( pckryInstance, laidOutItems ) { //...
// or with jQuery
$container.packery( 'on', 'layoutComplete', function( pckryInstance, laidOutItems ) {} )
  • pckryInstance

    Type: Packery

    the Packery instance

  • laidOutItems

    Type: Array of Packery.Items

    items that were laid out

Triggered after a layout and all positioning transitions have been completed.

pckry.on( 'layoutComplete', function( pckryInstance, laidOutItems ) {
  console.log('Packery layout completed on ' + laidOutItems.length + ' items');
});
